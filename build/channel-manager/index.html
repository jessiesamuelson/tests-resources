  • 1. Get all channels

          GET /channels

    Pagination: Customize results per page:

          ?per_page=[resultsPerPage]&page=[pageNumber]

    If page or per_page params is not specified, the default params are ?page=1&per_page=20, which means you get the first 20 results.

    Result object: 

          { channels: Array[20],
        pagination: {
          total: 382,
          page: 1,
          per_page: 20,
          num_pages: 20
      }

    To avoid pagination, add ?pagination=false. The result will only contain: 

        { channels: Array[382] }

  • 3. Get channels by categories

    There are two ways. Via channels:

          GET /channels?category=[categoryName]

    To get all channels from a group of categories, simply add comma separated category names. For example: 

          /channels?category=curated,news

    To get channels that belong to two categories, add plus separated category names. For example: 

          /channels?category=curated+news

    
    Via categories:
    GET /categories?name=[categoryName] /categories?name=curated,news /categories?name=curated+news

    The same pagination rules apply to /channels. /categories?name=[name] does not have pagination.

  • 3. Search for channel

    Search by name:

          GET /channels?term=[searchTerm]

    Search guidelines:

  • To get channels by name, simply input all or part of the name (not case-sensitive). For example: bark park.
  • category:curated gets all channels with category Curated.
  • To get all channels from multiple categories, add comma separated categories category:curated,news
  • To get all channels that belong to two or more categories, add plus separated categories category:curated+news
